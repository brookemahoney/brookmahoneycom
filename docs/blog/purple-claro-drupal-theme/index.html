Purple Claro Drupal Theme

I made a simple sub-theme of Claro that I use for this site called Purple Claro. The only difference is it adds some shades of purple to it. When I'm doing contrib work, I end up seeing the grey Claro theme from core but I miss my Purple Claro. So, I planned to publish my purple theme so it's easy to pull it in with composer when I'm doing other work.

My first thought was to make a sub-theme of Claro. I tried that. But it turns out that when you just try to make a simple sub-theme using the instructions on Drupal.org, there are things that still didn't work right. I had to manually export config for block layout, for starters. Then, I noticed that the nice little star icon next to page titles to add a shortcut was missing. So, I looked into how that was added but it turns out that the shortcut module and the Claro theme are dependent on each other for that functionality making it so I can't just reuse it. There seem to be a lot of modifications like this in Claro that I would have to figure out workarounds for.

So, my next thought is to make a module that just injects some CSS into Claro instead of making a sub-theme. We'll see how that goes...

OK, it's the next day. The dev version of the new Purple Claro module is on Drupal.org :)
